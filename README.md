# Keymap
## General Keymaps

| Keymap            | Function                                      |
| :----------------:| ----------------------------------------------|
| `;`               | Map leader                                    |
| `ss`              | Horizontal split (S split)                    |
| `sv`              | Vertical split                                |
| `Alt + Arrow key` | Resize panel                                  |
| `//`              | Search highlighted text                       |
| `/\`              | Turn off highlight                            |
| `[b`              | Go to previous buffer                         |
| `]b`              | Go to next buffer                             |
| `<leader>tm`      | Toggle Markdown Preview                       |
| `<leader>bd`      | Close buffer without exiting Neovim           |
| `<F2>`            |  Toggle Nerdtree                              |

## coc.nvim Keymaps

| Keymap            | Function                                      |
| :----------------:| ----------------------------------------------|
| `<leader>rn`      | Rename value                                  |
| `<leader>qf`      | Quick fix                                     |
| `gd`              | Go to definition                              |
| `gi`              | Go to implementation                          |
| `gy`              | Go to type definition                         |
| `gr`              | Go to references                              |
| `[g`              | Diagnostic previous                           |
| `]g`              | Diagnostic next                               |
| `K`               | Show documentation                            |
| `<leader>td`      | Toggle Coc diagnostic                         |
| `<leader>tb`      | Toggle buffer diagnostic                      |
| `<leader>ts`      | Toggle suggestion                             |
